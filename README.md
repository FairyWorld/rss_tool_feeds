## 💪 如何贡献

任何方式的贡献都对项目大有裨益。

1. 提交订阅更新，详细参见 [如何参与](https://feeddd.org/how-to-contribute)
2. 分享给更多人
3. 报告遇到的错误，创建 [issue](https://github.com/feeddd/feeds/issues)
4. 分享你的想法或建议，创建 [issue](https://github.com/feeddd/feeds/issues)
5. 给个星星 ⭐
